# DevOps-Two-Tier-Dockerized-Application-Deployment-AWS
Deploying a Two-Tier Dockerized Application to AWS.

Deploying a two-tier web application (WordPress + MySQL) to AWS. 
Deployment is to the cloud using Docker containers on AWS.
Code involves 2 parts: Setting up the AWS infrastructure using the AWS Management Console and using Linux, bash scripting, networking, and Docker to deploy and automate the application independently.
